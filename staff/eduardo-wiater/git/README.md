# git - 

## git init 

initializes a repo

```sh
$ git init
```

## git status

shows the status of r epo

```sh
$ git status
```

## git add

adds content or charges to be tracked

```sh
$ git add
```

## git commit 
creates an history 

```sh
$ git commit -m " ...texto"
```

## git log 
show the repo history of commits and branches

```sh
$ git log
```

### git log --graph
shows the history in vertical graph mode

```sh
$ git log --graph
```

## git config --list

```sh
$ git --list
```

## show the globing 
```sh
$ git config --list --global
```

## showing the diferences with the working copy

```sh
$ git diff
```

## adding all files

```sh
$ git add .
```
